Sum of Contiguous Subarray

Write a function that takes in a non empty array of integers and returns the maximum sum that can be obtained by summing up all the numbers in a nonempty subarray of the input array. A subarray must only contain adjacent numbers.

Input: [1,-3,2,1,-1]
Output: 3 [2,1]

Input: [5, 4, -9, 6, 7]
Output: 13 [6,7]

Input: [3, 5 , -9, 1,3] 
Output: 8 [3, 5]
